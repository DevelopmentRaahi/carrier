Role Description

This is a full-time on-site role for a Windows Server Administrator at DOMGYS in Noida. The server administrator will be responsible for the day-to-day tasks associated with server administration, system administration, troubleshooting, technical support, and operating systems.

  • The admin will be responsible for overall maintenance, platform upgrades.
  • Majorly responsible for provisioning, creation, troubleshooting of Windows VPS and cloud servers and related issues
  • Should have good hands-on knowledge of server migration along with data and applications with secure systems
  • Should have good knowledge of DNS establishment and creation of DNS server.
  • Must to have good information on records creation in DNS
  • Must have worked on data base (MYSQL, SQL)
  • Configuration to ensure users have proper access to files within the database.
  • They will work with support team members to ensure that the platform is running smoothly and efficiently.
  • You will also be responsible for troubleshooting any issues that arise with the platform and ensuring that it is secure.
  • This resource will serve as an individual contributor within the organization.
  • You will be the senior escalation point for any R platform administration related issues and must showcase the ability to lead/guide supporting team members.
Qualifications
  • 2-3 years' experience in Windows server administration.
  • Having experience with expertise in user provisioning (i.e. Active Directory, SSO, User Authentication, etc.)
  • Server Administration, System Administration, and Troubleshooting Skills
  • Technical Support Experience.
  • Experience with Multiple Operating Systems.
  • Strong knowledge of Windows systems.
  • Bachelor's Degree or Higher in Computer Science or Related Field.)
  • Excellent Communication Skills.
  • Teamwork and Collaborative Skills.
  • Problem-solving and Analytical Skills.
  • Ability to work independently, guide supporting team members and contribute to team meetings.
